# patient-health-monitoring-system
Arduino-based Patient Health Monitoring System using LM35 temperature sensor and heart rate sensor to monitor vital health parameters and provide real-time alerts through LED and buzzer.
## Components Used
- Arduino Uno
- LM35 Temperature Sensor
- Heart Rate Sensor
- LED
- Buzzer
- Breadboard
- Jumper Wires

## Features
- Temperature Monitoring
- Heart Rate Monitoring
- Real-Time Alerts
- Serial Monitor Output

## Technologies Used
- Arduino IDE
- Embedded C
- Electronics
